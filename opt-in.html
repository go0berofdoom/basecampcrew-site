BaseCamp Family Invites

BaseCamp Family Invites is a transactional SMS-only program. Messages are sent from our verified toll-free number solely to deliver a family invite requested by an existing BaseCamp user.

One invite. Exactly one message.Message frequency is one message per invite. We never send recurring or marketing messages through this program.

Sending numberVerified toll-free number

Express-consent workflow

Before the app can send an invite, the inviting user must affirm that the recipient expressly agreed to receive the one-time text. The required checkbox is unchecked by default, and the “Send invite” button remains disabled until it is checked. This workflow documents the express consent required under Twilio rejection code 30446.

  1. An existing BaseCamp user enters their family member’s phone number.
  2. The user affirms the recipient’s consent by checking the required consent checkbox.
  3. Exactly one invite message is sent.
  4. The recipient can reply STOP at any time and HELP for help; STOP is honored automatically and permanently.

Message frequency

One message per invite; no recurring or marketing messages, ever. Message and data rates may apply.

Opt-out & help

Reply STOP to unsubscribe, HELP for assistance, or email hello@basecampcrew.com. STOP requests are honored automatically and permanently.
